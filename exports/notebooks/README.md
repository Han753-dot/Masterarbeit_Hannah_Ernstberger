# Notebook-Exporte

Diese Notebooks erzeugen die exportierten Tabellen, Abbildungen und Metadaten. Sie enthalten den vollständigen Ausführungscode und benötigen keine separaten `.py`-Skripte.

| Notebook | Zweck |
| --- | --- |
| `01_finale_tabellen_und_abbildungen.ipynb` | Zentrale Tabellen, Detailtabellen und Ergebnisabbildungen. |
| `02_screenshot_beispielabbildungen.ipynb` | Screenshot-Beispielabbildungen, Kandidatenvarianten, Metadaten und Frame-Kopien. |
| `03_modellbewertung_textbeispiele.ipynb` | Kombinierte Abbildung zu gut und schwächer bewerteten Modellannotationen. |
| `04_influencerinnen_auswahl.ipynb` | Influencerinnen-Auswahlabbildung. |
| `05_fazit_beispielabbildungen.ipynb` | Beispielabbildungen zu zentralen Fazitpunkten. |
| `06_visuelle_oberkategorien_beispielabbildungen.ipynb` | Beispielabbildungen zu visuellen Oberkategorien. |
